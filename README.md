# Mini-LMS (Learning Management System)
> A full-stack online learning platform built with React, Flask, and SQLite

## Project Overview
Mini-LMS is a simplified Learning Management System that allows:
- **Instructors** to create courses, post lessons, and assign homework
- **Students** to enroll in courses, view content, and submit assignments

## Tech Stack

**Frontend:**
- React
- HTML/CSS
- Deployed on Vercel

**Backend:**
- Python (Flask)
- SQLite Database
- Deployed on Render
